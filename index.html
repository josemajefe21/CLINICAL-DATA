Sistema Integral de Gestión Médica

👨‍⚕️ ¿Qué puede hacer este software?

👤 Gestión Completa de Pacientes

  • Registro de datos personales completos
  • Búsqueda por nombre, DNI o diagnóstico
  • Historial médico completo
  • Estadísticas automáticas de cada paciente

🩺 Sistema de Visitas Médicas

  • Consulta: Evaluación del paciente
  • Seguimiento: Control y evolución del tratamiento
  • Urgencia: Atención médica urgente
  • Control: Revisiones programadas

📊 Análisis y Reportes

  • Gráficos de evolución del estado clínico
  • Estadísticas de visitas y tratamientos
  • Exportación de historias clínicas en PDF
  • Respaldo de datos en formato JSON

⏰ Sistema de Recordatorios

  • Notificaciones automáticas del navegador
  • Recordatorios por paciente
  • Clasificación por urgencia (urgentes, hoy, pendientes)
  • Gestión completa de tareas médicas

📅 Calendario de Citas

  • Vista de calendario mensual interactiva
  • Programación de citas por paciente
  • Indicadores visuales de días ocupados
  • Gestión completa de agenda médica

🔐 Seguridad y Sincronización

  • Autenticación segura con Firebase
  • Sincronización automática en la nube
  • Datos protegidos por usuario
  • Acceso desde cualquier dispositivo

✨ Diseño Moderno y Responsive

Interfaz elegante con modo nocturno, efectos glassmorphism y optimizada para móviles
